## 技能名称
GitHub 热门仓库每日分析助手

## 功能介绍
自动拉取 GitHub 近 7 天新建高分热门仓库，进行多维度数据分析、分类统计，生成精美可视化报告。
支持自定义指定编程语言查询，自带每日 09:00 定时推送，自动归档报告到本地日志文件夹。

## 触发指令
- GitHub热点
- 今日热门仓库
- 热门仓库
- GitHub trending
- Python的GitHub热点
- Go语言热门仓库

## 特色能力
1. 自动统计仓库星标、Fork、平均数据
2. 语言分布图形展示、仓库智能分类
3. AI/Web/后端/工具类等多维度归类
4. 提取热门技术标签、展示最新创建仓库
5. 详细仓库详情、简介、开源协议、地址汇总
6. 定时每日9点自动执行，自动保存TXT文档
7. 支持手动指令随时查询，一键生成报告

## 使用方式
1. 直接发送关键词：今日热门仓库
2. 指定语言查询：Java的GitHub热点
3. 程序常驻运行，每日早上9点自动推送日报
4. 所有报告自动保存到 github_trending_logs 目录

## 依赖环境
Python3.8+
安装依赖：pip install -r requirements.txt
